## Задание 2

Были выполнены следующие виды сканирования:

- SYN (`nmap -sS`)
- FIN (`nmap -sF`)
- Xmas (`nmap -sX`)
- UDP (`nmap -sU`)

### Чем отличаются эти режимы сканирования?

**SYN**

Отправляется только SYN-пакет. Если порт открыт, сервер отвечает SYN/ACK. Соединение полностью не устанавливается, поэтому такой способ считается быстрым.

**FIN**

Отправляется пакет с флагом FIN. Если порт закрыт, сервер отвечает RST. Если порт открыт, обычно ответа нет, поэтому Nmap показывает состояние `open|filtered`.

**Xmas**

Отправляются пакеты с флагами FIN, PSH и URG. Работает почти так же, как FIN-сканирование.

**UDP**

Проверяются UDP-порты. Если порт закрыт, сервер обычно отвечает сообщением ICMP Port Unreachable. Если ответа нет, порт считается открытым или фильтруемым.

### Как отвечает сервер?

- При SYN-сканировании сервер отвечает SYN/ACK для открытых портов.
- При FIN и Xmas-сканировании закрытые порты отвечают RST, а открытые обычно не отвечают.
- При UDP-сканировании закрытые порты отвечают ICMP Port Unreachable, а открытые могут отвечать своим протоколом или не отвечать совсем.
